## Tech stack

- Node.js
- Express.js
- MongoDB
- Mongoose

## Routes

### GET routes

`/`

`/cards`

`/choirs`

`/concerts`  
`/concerts/:id`

`/cv`  
`/cv/:year`

`/discography`  
`/discography/:number`

`/memories`  
`/memories/authors`  
`/memories/:id`

`/orchestras`

`/pages`  
`/pages/:path`

`/pictures`  
`/pictures/:id`

`/premieres`

`/repertoire`

### POST route

`/send-message`
